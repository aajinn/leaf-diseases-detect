Theme Controls

Click buttons to test different background themes:

Animation Info

Leaves: Green particles representing healthy plants

Bacteria: Red particles representing diseases

Collision: Particles bounce and glow when they collide

Themes: Background color changes based on page state

Theme Descriptions

Guest: Light blue - Before login pages
User: Light green - Logged in user pages
Admin: Blue tint - Admin panel
Analyzing: Yellow/orange - During image analysis
Healthy: Vibrant green - No disease detected

How to Use

  1. Watch the leaves and bacteria float across the screen
  2. Click theme buttons to see color changes
  3. Click "Test Flow" to simulate an analysis sequence
  4. Notice how particles collide and bounce off each other
