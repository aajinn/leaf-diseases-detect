# Technology Stack

## Frontend
| | Technology | Role |
|---|---|---|
| **Language** | HTML5 + JavaScript (ES6+) | Structure & logic |
| **Styling** | Tailwind CSS 3.4 | UI framework |
| **Camera** | Browser MediaDevices API | Live leaf capture |
| **PDF** | jsPDF | Prescription export |

---

## Backend
| | Technology | Role |
|---|---|---|
| **Language** | Python 3.9+ | Core language |
| **Framework** | FastAPI | REST API & routing |
| **Server** | Uvicorn | ASGI server |
| **Database** | MongoDB + Motor | Data storage (async) |
| **Auth** | JWT + bcrypt | Login & security |
| **Payments** | Razorpay | Subscription billing (INR) |

---

## AI / External APIs
| | Service | Role |
|---|---|---|
| **Vision AI** | Groq API — Meta Llama 4 Scout 17B Vision | Leaf disease detection from images |
| **Recommendations** | Perplexity AI | YouTube treatment video suggestions |
