- **7.6.1 Level 0 Context Diagram** — System boundary; external entities: Guest, User, Admin, Enterprise User, API Client, Groq API, Perplexity API, Razorpay, MongoDB.
- **7.6.2 Level 1 Admin** — Admin panel: dashboard, user management, API config, subscription stats, enterprise API keys, system settings, usage charts, prescription analytics.

- **7.6.5 Level 1 Enterprise User** — Enterprise subscriber: enterprise dashboard, bulk analysis, analytics, API key management, CSV export, analysis history, usage stats.
